## Letöltéshez a célmappában meg kell nyitni a cmd-t, és kiadni a következő parancsot

```
git clone https://github.com/Adri959/teszt5555

```
## Virtuális környezet létrehozása

```
py -m venv .

```
## Aktiválás

```
Scripts\activate

```
## A requirements.txt tartalmának telepítésa

```
pip install -r requirements.txt

```
## Belépés a backend mappába

```
cd backend

```
## Futtatás

```
py manage.py runserver

```
